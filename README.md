# Calories Coutner
## Introduction
Calories counter is a web application that helps you track your daily calorie and macronutrient intake to achieve your fitness goals with ease.

The app provides a convenient and intuitive UI where you can:
- Track your daily food intake
- Monitor calories, protein, carbs and fats
- Set your current and goal weight
- Get a personalized diet plan tailored to your needs

In short: less guessing, more progress

## Tech Stack
- **Frontend:** Vue.js + Vite + TanStack Query
- **Backend:** Next.js API routes
- **Database:** Neon Postgres
- **ORM:** Drizzle ORM + Zod

## Installation
Clone the repository:

```sh
git clone https://github.com/timurenk0/app-web.git
```
Open the project in oyur IDE.

Install dependencies from the root directory of the project.

```sh
cd frontend && npm install
cd backend && npm install
```

## How to Run the Project
From the root of the project, open **two terminals** and run:

```sh
npm run front
```

```sh
npm run back
```

Then open the frontend in your browser:
```sh
http://localhost:5173
```
